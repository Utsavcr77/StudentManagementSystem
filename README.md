# 🎓 Student Management System

A **console-based Student Management System** built in C++ with persistent file storage and a clean menu-driven interface.

---

## 📋 Features

| Feature | Description |
|---|---|
| ➕ Add Student | Add student with ID, name, age, course, and GPA |
| 📋 Display All | View all students in a formatted table |
| 🔍 Search | Find any student by their unique ID |
| ✏️ Update | Modify any field of an existing student |
| 🗑️ Delete | Remove a student with confirmation prompt |
| 📊 Statistics | View total count, average/highest/lowest GPA |
| 💾 Persistent Storage | All data saved to `students.dat` automatically |

---

## 🛠️ Tech Stack

- **Language:** C++17
- **Concepts:** File I/O (`fstream`), Structs, Vectors, STL algorithms, Input validation

---

## 🚀 How to Compile & Run

### Linux / macOS
```bash
git clone https://github.com/Utsavcr77/StudentManagementSystem.git
cd StudentManagementSystem
make run
```

### Windows (MinGW)
```bash
g++ -std=c++17 -Wall -o sms.exe main.cpp
sms.exe
```
